Links internos

Para inserir um link interno tenha os arquivos html na mesma pasta. Uliliza-se a tag
< a href="arquivo da página*" rel="next"> texto clicável*< /a> para a próxima página ou
< a href="arquivo da página*" rel="prev"> texto clicável* < /a> para a página anterior

Caso a página para a qual deseja voltar esteja em outra pasta insira "../" antes do nome do arquivo. Exemplo: < a href="../index. html"> texto clicável* < /a>.

Links para download

Download de PDF

Insira em href="" o local do arquivo que deseja disponibilizar para download.

< a href="arquivos-pdf/primeiros-passos-musica-classica.pdf">Arquivo png< /a>

Adicione os parâmetros download="" e type="". Exemplo;
< a href="arquivos-pdf/primeiros-passos-musica-classica.pdf" download="arquivos-pdf/primeiros-passos-musica-classica.pdf" type="application/pdf">Arquivo png< /a>

Imagens Dinâmicas

Tamanhos diferentes de imagens para serem usadas de acordo com o tamanho da tela: 300x300 para celular, 700x700 para tablet e 1000x1000 para Pc, como exemplo

A tag < picture>< /picture> permite o uso de diferentes sources. Recomenda-se iniciar inserindo a imagem de maior tamanho e incluir as demais em ordem decrescente, logo acima da anterior.

Dentro de < picture>< /picture> adicione a tag < img src="arquivo da imagem*" alt="descrição da imagem">. Logo acima desta tag adicione a tag source:media

Modifique source media="(min-width: )" para max. Adicione em max-width: o tamanho da imagem a ser carregada, somando 50px. Exemplo: < source media="(min-width:1050px).

O navegador irá carregar por padrão a imagem maior. Caso a tela tenha o tamanho menor que o estabelecido em max-widht, será carregada a imagem seguinte. Em srcset="" adicione o arquivo desta imagem. Logo em seguida adicione no parâmetro type="" o media type correspondente ao arquivo.

Para inserir mais tamanhos repita o processo, adicionando source logo acima.

< picture>
< source media="(min-width: )" srcset="" type="image/">
< source media="(min-width: )" srcset="" type="image/">
< img src="" alt="">
< /picture>

Áudio

Utilize a tag < audio>< /audio>
< audio src="audios/drive-breakbeat-173062.mp3 " controls>< /audio>

Múltiplos formatos de áudio

Para ter maior compatibilidade com os navegadores utiliza-se vários formatos de Áudio: MP3, OGG e WAV. Adicione os diferentes arquivos da seguinte forma:

< audio>
< source src="audio.mp3" type="audio/mpeg">
< source src="audio.ogg" type="audio/ogg">
< source src="audio.wav" type="audio/wav"> São muitos pesados, recomenda-se não usar.
< /audio>

Outro parâmetro: preload="metadata", carrega as informações do arquivo de áudio.

Vídeos

Handbrake: Ferramenta gratuita para converter os vídeos.

Formatos de vídeo: MP4 e MP4V, WEBM E OGG (até 1080p)

utiliza-se a tag < video src="videos/video-html.mp4" width="600" controls poster="imagens/paisagem-650.jpeg">< /video>

width é a largura do vídeo; não influencia no tamanho do arquivo.
Conntrols exibe os controles para tocar o vídeo

Para incluir vários formatos de vídeo utliliza-se os source:src.

O parâmetro poster="" permite adicionar um arquivo de imagem para ser a capa do seu vídeo.

Adicione um parágrafo com uma mensagem, logo após os sources. Ele aparecerá caso os nenhum dos arquivos seja compatível com o navegador.

Inserir vídeos do Youtube

Com o vídeo desejado já aberto no youtube, clique em COMPARTILHAR, depois em INCORPORAR e em seguida em COPIAR. Após, é só COLAR no editor de código.
